Zen Cart v2 French language pack documentation


This language pack is an update for Zen Cart® version 2.0.0 and above of old French language files from v1.5.4 and v1.5.7 plus added translations. It contains French language files for admin, catalog, plugins and installer.

Install

Before starting there are few  things to do or check.

Don't forget, this language pack is for Zen Cart 2.0.0 and above! It is not compatible with older versions.

If you have a cart already installed, you must do a complete backup including database. For files a copy is easy, but for database you must export it to an SQL file you can import again later if necessary. This is a very important step! In case of troubles, it will save you lot of time and more.
Please, choose from below menu to get detailed explanations:

New cart installation

You must prepare a server space, domain and database as you would do for a standard Zen Cart® install.

Extract Zen Cart latest version (v2.1.0) you have just downloaded and copy all files in your site folder. Extract Zen Cart French Language Pack too and copy admin, includes, zc_install and zc_plugins folders from Files_to_Install folder (or Files_to_Install_for_v1.5.8 folder for Zen Cart v1.5.8a) to your cart.

Now you should open your browser and in address bar type your site address using your real domain in place of 'yoursite' and http or https depending on your case:
https://yoursite.com/zc_install/ (or https://yoursite.com/ and click on 'Click here' link)

Zen Cart installer will start in English, you can change to French, and do some system checks. If there is no big problem with system check, you can proceed to install like a normal Zen Cart by clicking on 'Continue' button. After your first login in admin, installation is done!

Now you can go to configuration chapter to get ready for sale.

Already installed cart

When installing over an existing Zen Cart, it is not much more difficult than a new install.

First, Extract Zen Cart French Language Pack and rename admin folder to the same name as your running Zen Cart®. Then copy folders admin (renamed to your real admin folder name), includes and zc_plugins from Files_to_Install folder (or Files_to_Install_for_v1.5.8 folder for Zen Cart v1.5.8a) to your cart.

If you have some plugins installed, you must add French language folder and files to their language folder. If you don't have the French translation for these plugins, just copy 'english' folder and rename the copy to 'french'. If these files are missing, Zen Cart® might default to English or possibly crash when trying to use French language. After that, you will have time to do the French translation for these plugins later.

Now you must add French language to Zen Cart through 'admin>Localization>Languages'. This has to be done manually through admin interface as it sets everything necessary for a new language in Zen cart.

Name: French
Code: fr
Image: icon.gif
Directory: french
Sort Order: 0

Set French as default if you want. Then import/run SQL queries in file '../sql/orders_status_update.sql' (provided with the language pack) to update order status table with French. Finally, import French zones into database using file '../sql/zones_francaises.sql'.

Configuration

You might want to set  Euro as default currency (don't forget to update exchange rates), add a new tax class (VTA) and rate, set your store location and postal code, define some geo-zones for your shipping modules and set these modules and payment ones too according to these zones.
Please have a look at Zen Cart documentation, starting by here: https://docs.zen-cart.com/user/new_user_topics/

Recommended plugins

Language/Currency Links in Header

This plugin should be in any multilingual Zen Cart. Having language switch in header is very useful and practical especially on mobile displays.

Postcode auto fill

This plugin was designed to help customers to fill create account or new address form. It simply auto fill state, city and sometime part of street address, when customer entered is postal code. It is not a necessary feature, but avoid errors like inconsistent address where postal code and address do not correspond. Drawback is that postal code databases are big and plugin had to be separated in two parts. Importing big files in MySQL with phpMyAdmin could be problem too if PHP and MySQL are not set properly.

Image Handler 5 (Zen Cart 1.5.7+)

This plugin enhances Zen Cart images by adding a handler to offer automatic resizing of images, caching thumbnails, and more. You can upload multiple sizes of an image from the Admin image handler page.

Bonus content

You will find some language files translation for few plugins in folder '.../optional plugins/Encapsulated' for encapsulated plugins and '.../optional plugins/Non-Encapsulated' for non-encapsulated plugins. To use these, just copy files as follow:

For encapsulated plugins, copy the whole plugin folder in your cart 'zc_plugins' folder.
For non-encapsulated plugins, copy the 'includes' and 'admin' folders to your cart root, but you must rename the 'admin' folder to your actual admin folder name before.

Be aware that some plugins like Bootstrap Japanese version have customized files other than language files and you might need to merge those rather than copy them.

Uninstall

Set default language to English or whatever you want and delete French language from Zen Cart Admin menu 'Localization>Languages'.
Then you can delete all French language files in language folders '../french' folders or just let them where they are if you plan to re-install French language later.
Don't forget to remove any admin setting you did specifically for French.

Documentation pour Zen Cart v2 French language pack


Ce pack linguistique est une mise à jour pour Zen Cart® version 2.0.0 et supérieure des anciens fichiers de langue française des versions 1.5.4 et 1.5.7, ainsi que des traductions ajoutées. Il contient des fichiers en français pour l'administration, le catalogue, les plugins et le programme d'installation.

Instalatlion

Avant de commencer, il y a quelques points à vérifier.

N'oubliez pas que ce pack linguistique est destiné à Zen Cart 2.0.0 et supérieur ! Il n'est pas compatible avec les anciennes versions.

Si vous avez une boutique déjà installée, vous devez effectuer une sauvegarde complète incluant la base de données. Pour les fichiers, une copie est facile, mais pour la base de données, vous devez l'exporter vers un fichier SQL que vous pourrez réimporter ultérieurement si nécessaire. Ceci est une étape très importante! En cas de problème, cela vous fera gagner beaucoup de temps et bien plus encore.

Veuillez choisir dans le menu ci-dessous pour obtenir des explications détaillées :

Nouvelle installation

Vous devez préparer un espace serveur, un domaine et une base de données comme vous le feriez pour une installation Zen Cart® standard.

Décompressez Zen Cart v2.0.0 ou plus récent que vous venez de télécharger et copiez tous les fichiers dans le répertoire de votre nouveau site. Décompressez également le pack de langue française et copiez les dossiers admin, include, zc_install et zc_plugins qui sont dans le répertoire Files_to_Install (ou Files_to_Install_for_v1.5.8 pour Zen Cart v1.5.8a).

Vous devez maintenant ouvrir votre navigateur et dans la barre d'adresse, saisissez l'adresse de votre site en utilisant votre domaine réel à la place de « votresite » et http ou https selon votre cas :
https://yoursite.com/zc_install/

Le programme d'installation de Zen Cart démarrera en anglais (vous pourrez passer au français) et effectuerq quelques vérifications du système. S'il n'y a pas de gros problème avec la vérification du système, vous pouvez procéder à l'installation comme avec un Zen Cart normal en cliquant sur le bouton « Continuer ». Après votre première connexion en admin, l'installation est terminée !

Vous pouvez maintenant aller au chapitre de configuration pour vous préparer à la vente.

Boutique déjà installée

Lors de l'installation sur un Zen Cart existant, ce n'est pas beaucoup plus difficile qu'une nouvelle installation.

Tout d’abord, décompressez le pack de langue française et renommez le dossier admin avec le même nom que celui de votre Zen Cart® en cours d’utilisation. Copiez ensuite sur votre site les dossiers admin (renommé en votre vrai nom de dossier administrateur), include et zc_plugins qui sont dans le répertoire Files_to_Install (ou Files_to_Install_for_v1.5.8 pour Zen Cart v1.5.8a).

Si des plugins sont installés, vous devez ajouter le dossier et les fichiers de langue française à leur dossier de langue. Si vous n'avez pas la traduction française de ces plugins, copiez simplement le dossier « english » et renommez la copie en « french ». Si ces fichiers sont manquants, Zen Cart® utilisera l'anglais par défaut ou éventuellement plantera lorsque vous essayerez d'utiliser la langue française. Après cela, vous aurez le temps de faire la traduction française de ces plugins plus tard.

Vous devez maintenant ajouter la langue française à Zen Cart via 'admin>Localisation>Langues'. Cela doit être fait manuellement via l'interface d'administration car elle configure tout ce qui est nécessaire pour une nouvelle langue dans Zen Cart®.

Name: French
Code: fr
Image: icon.gif
Directory: french
Sort Order: 0

Définissez le français par défaut si vous le souhaitez. Ensuite, importez/exécutez les requêtes SQL dans le fichier '../sql/orders_status_update.sql' (fourni avec le pack de langue) pour mettre à jour la table d'état des commandes avec le français. Finalement, importer les zones françaises dans la base de données en utilisant le fichier '../sql/zones_francaises.sql'.

Configuration

Vous souhaiterez peut-être définir l'euro comme devise par défaut (n'oubliez pas de mettre à jour les taux de change), ajouter une nouvelle classe de taxe (VTA) et un nouveau taux, définir l'emplacement de votre magasin et votre code postal, définir des zones géographiques pour vos modules d'expédition et définir également ces modules ainsi que ceux de paiement en fonction de ces zones. . Veuillez consulter la documentation Zen Cart, en commençant par ici : https://docs.zen-cart.com/user/new_user_topics/

Plugins recommandés

Language/Currency Links in Header

Ce plugin devrait être dans n'importe quel Zen Cart multilingue. Avoir le changement de langue dans l’en-tête est très utile et pratique notamment sur les écrans mobiles.

Postcode auto fill

Ce plugin a été conçu pour aider les clients à remplir un formulaire de création de compte ou de nouvelle adresse. Il remplit simplement automatiquement le département, la ville et parfois une partie de l'adresse postale, lorsque le client a saisi son code postal. Ce n'est pas une fonctionnalité nécessaire, mais évite les erreurs telles qu'une adresse incohérente où le code postal et l'adresse ne correspondent pas. L'inconvénient est que les bases de données de codes postaux sont volumineuses et que le plugin a dû être séparé en deux parties. L'importation de gros fichiers dans MySQL avec phpMyAdmin peut également poser problème si PHP et MySQL ne sont pas correctement configurés.

Image Handler 5 (Zen Cart 1.5.7+)

Ce plugin améliore les images Zen Cart en ajoutant un gestionnaire pour proposer un redimensionnement automatique des images, la mise en cache des vignettes, etc. Vous pouvez télécharger plusieurs tailles d’image à partir de la page du gestionnaire d’images Admin.

Bonus

Vous trouverez quelques fichiers de traduction de langue pour quelques plugins dans le dossier '.../optional plugins/Encapsulated' pour les plugins encapsulés et '.../optional plugins/Non-Encapsulated' pour les plugins non encapsulés. Pour les utiliser, copiez simplement les fichiers comme suit :

Pour les plugins encapsulés, copiez l'intégralité du dossier du plugin dans le dossier « zc_plugins » de votre site.
Pour les plugins non encapsulés, copiez les dossiers « includes » et « admin » à la racine de votre site, mais vous devez auparavant renommer le dossier « admin » avec le nom réel de votre dossier d'administration.

Sachez que certains plugins comme la version japonaise de Bootstrap ont des fichiers personnalisés autres que des fichiers de langue et vous devrez peut-être les fusionner plutôt que de les copier.

Uninstall

Définissez l'anglais comme la langue par défaut ou celle de votre choix et supprimez la langue française à partir du menu administrateur de Zen Cart « Localisation > Langues ».
Ensuite, vous pouvez supprimer tous les fichiers de langue française dans les dossiers de langue « ../french » ou simplement les laisser là où ils se trouvent si vous envisagez de réinstaller la langue française ultérieurement.
N'oubliez pas de modifier tout paramètre d'administration que vous avez ajusté spécifiquement pour le français.
